# 郑州热门学校咨询站

这是一个纯静态演示站，不需要服务器。

## 直接预览

把下面 4 个文件放到仓库里的 `schools/` 目录即可：

- `schools/index.html`
- `schools/styles.css`
- `schools/script.js`
- `schools/README.md`

## GitHub Pages

1. 把代码推到 GitHub 仓库
2. 在仓库 `Settings -> Pages`
3. `Source` 选择当前分支
4. 如果你希望直接打开这个页面，建议把站点部署目录指向仓库根目录，访问时进入 `/schools/`
5. 保存后等待几分钟即可生成访问链接

## 当前版本包含

- 郑州热门初中、小学展示
- 学段筛选
- 区域筛选
- 关键词搜索
- 每个学校卡片下方的咨询菜单
